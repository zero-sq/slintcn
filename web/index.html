Why slintcn

A distribution system, not a library.

The shadcn model, ported to Slint: a CLI + registry that copies source into your repo. You own every file.

Own the code

`slintcn add` copies .slint source into your project — edit freely, no lock-in. Imports auto-rewrite to your layout.

A real CLI + registry

init · add · list · view · build. Install from the bundled registry, a remote URL, or a custom `@ns/name`.

Dark-glass + theming

Measured-from-shadcn tokens. Pick a base color (neutral · zinc · slate · stone) at init; runtime dark/light.

37 components

Buttons to Dialogs to Tables to Tooltips — typed enum variants, focus rings, keyboard nav, animations.

5 blocks
